# SaaS 가격 정책 수립 시스템

AI 기반 SaaS 제품의 가격 정책을 수립하는 웹 애플리케이션입니다. Claude API를 활용하여 비용 분석, 시장 조사, 가격 모델 설계, 재무 시뮬레이션을 자동으로 수행합니다.

## 기능

- 📊 **비용 분석**: 변동비와 고정비를 기반으로 규모별 비용 계산 및 BEP 분석
- 🔍 **시장 조사**: 경쟁사 가격 벤치마킹 및 시장 포지셔닝 분석
- 💡 **가격 모델 설계**: 2-4개의 Tier 설계 및 가격 앵커링 전략
- 💰 **재무 시뮬레이션**: 12개월 매출/비용 예측 및 시나리오 분석
- 📥 **보고서 다운로드**: 전체 분석 결과를 Markdown 파일로 내보내기

## 설치 방법

### 1. 사전 요구사항

- Python 3.8 이상
- Anthropic API Key ([발급받기](https://console.anthropic.com/))

### 2. 패키지 설치

```bash
# 가상환경 생성 (선택사항)
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate

# 필요한 패키지 설치
pip install -r requirements.txt
```

### 3. API 키 설정

`.env.example` 파일을 `.env`로 복사하고 API 키를 입력하세요:

```bash
copy .env.example .env
```

`.env` 파일을 열어서 API 키를 입력:

```
ANTHROPIC_API_KEY=your_actual_api_key_here
```

또는 앱 실행 후 사이드바에서 직접 입력할 수도 있습니다.

## 실행 방법

```bash
streamlit run app.py
```

브라우저가 자동으로 열리며 `http://localhost:8501`에서 앱이 실행됩니다.

## 사용 방법

1. **API 키 입력**: 왼쪽 사이드바에서 Anthropic API 키를 입력합니다
2. **정보 입력**: 다음 항목들을 입력 폼에 작성합니다:
   - 제품/서비스 개요
   - 비용 구조 (변동비, 고정비)
   - 예상 사용 패턴
   - 타겟 고객층
   - 사업 목표
   - 제약사항
3. **분석 시작**: "가격 정책 분석 시작" 버튼을 클릭합니다
4. **결과 확인**: 4단계 분석이 자동으로 진행되며, 탭을 통해 각 단계별 결과를 확인할 수 있습니다
5. **보고서 다운로드**: "전체 보고서 다운로드" 버튼으로 분석 결과를 저장합니다

## 주요 컴포넌트

### Main Agent
전체 워크플로우를 조율하고 최종 권고안을 작성합니다.

### Skill 1: 비용 분석기
- 사용자 규모별 총 비용 계산
- Break-even point 도출
- 비용 민감도 분석

### Skill 2: 시장 가격 조사기
- 경쟁사 가격 벤치마킹 (웹 검색)
- 시장 가격대 파악
- 가격 모델 트렌드 분석

### Skill 3: 가격 모델 설계기
- 2-3개의 가격 Tier 설계
- 각 Tier별 기능/제한 정의
- 가격 앵커링 전략 제안

### Skill 4: 재무 시뮬레이터
- 12개월 매출/비용/이익 예측
- MRR, ARR, ARPU, LTV, CAC 계산
- 낙관/기본/비관 시나리오 분석

## API 사용량

한 번의 전체 분석에 약 40,000-60,000 토큰이 소비됩니다:
- Skill 1-4: 각 8,000 토큰 (총 32,000)
- 최종 권고: 8,000 토큰
- 입력 토큰: 약 10,000-20,000

Claude Opus 4.5 기준 비용: 약 $0.50-0.80 per analysis

## 문제 해결

### API 키 오류
- `.env` 파일에 올바른 API 키가 있는지 확인
- 또는 앱 사이드바에서 직접 입력

### 프롬프트 로드 실패
- `saas_pricing_prompts.md` 파일이 동일 디렉토리에 있는지 확인

### 분석 중 오류
- API 키의 권한 및 잔액 확인
- 인터넷 연결 상태 확인 (시장 조사 단계에서 웹 검색 사용)
